# CRM Automation Workflow

## Overview
This project demonstrates a simulated CRM automation workflow.

## Workflow
- New lead is received
- Data is processed and stored
- Automatic email notification is sent
- Lead status is updated

## Concepts Used
- API integration
- Automation workflows
- Data processing
- Event-driven actions

## Purpose
Simulate how CRM systems can automate customer onboarding and communication, reducing manual work and improving efficiency.
